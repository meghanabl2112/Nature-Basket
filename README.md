
# Nature Basket
Deliver fresh produce from farmers to buyers, with delivery tracking.

## Tech Stack
- Frontend: React (Vite)
- Backend: FastAPI (Python)
- Database: MySQL
- Dev setup: Docker Compose

## Quickstart
1. cp backend/.env.example backend/.env
2. docker compose -f infra/docker-compose.yml up --build
3. Open http://localhost:5173 (frontend) and http://localhost:8000/docs (backend API)
